# Plugin: [Neogen](https://github.com/danymat/neogen)

## Description

Neogen is an annotation generator for functions and classes.
It helps you generate documentation automaticallyâ€”no more manual docstring writing.

## Requirements

- Tree-sitter parsers must be installed

## Features

- Generates clean and well-structured documentation snippets
- Supports automated documentation generation for codebases

- Currently configured to use **Google-style docstrings** for **Python**


## Commands

The following commands are provided by Neogen.
They are typically mapped to keybindings for faster access.

## Key Bindings

| Keybinding       | Command            | Description                       |
|------------------|--------------------|-----------------------------------|
| `<leader>nc`     | `Neogen class`     | Generate docstring for a class    |
| `<leader>nf`     | `Neogen func`      | Generate docstring for a function |
| `<leader>nm`     | `Neogen file`      | Generate docstring for a file     |
| `<leader>nt`     | `Neogen type`      | Generate docstring for a type     |
